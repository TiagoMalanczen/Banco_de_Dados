# 🗄️ Estudos de Banco de Dados Relacional (SQL)

Repositório dedicado ao armazenamento de scripts, laboratórios práticos e evolução conceitual em modelagem e manipulação de bancos de dados relacionais. Conteúdo alinhado à grade de Análise e Desenvolvimento de Sistemas (UTFPR).

## 🚀 Tecnologias e Ambientes
* **SGBD:** MariaDB / MySQL
* **Ambiente de Desenvolvimento:** VS Code (Database Client) & XAMPP Control Panel
* **Interface:** Terminal/PowerShell para testes de integridade

## 📌 Conteúdo Geral e Tópicos Estudados
O repositório está estruturado de forma incremental para consolidar os fundamentos teóricos e práticos da disciplina:
- **Modelagem Conceitual e Lógica:** Arquivos de diagramas, mapeamento de regras de negócio e integridade referencial.
- **Construção e Manipulação de Dados:** Scripts estruturados englobando a criação de esquemas, tabelas e restrições.
- **Consultas e Relatórios:** Implementação de buscas avançadas, junções, agrupamentos e funções agregadas.
